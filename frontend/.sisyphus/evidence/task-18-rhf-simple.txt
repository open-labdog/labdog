Task 18: Migrate 5 simple CRUD forms to React Hook Form + Zod validation
=========================================================================

Date: 2026-03-19

Files migrated:
  1. app/(dashboard)/groups/new/page.tsx       — useForm<GroupInput> + groupSchema
  2. app/(dashboard)/ssh-keys/page.tsx          — useForm<SshKeyInput> + sshKeySchema (create dialog)
  3. app/(dashboard)/git-repos/page.tsx         — useForm<GitRepoInput> + gitRepoSchema (create + edit dialog)
  4. app/(dashboard)/hosts/new/page.tsx         — useForm<HostInput> + hostSchema
  5. app/(dashboard)/hosts/discover/page.tsx    — useForm<CidrInput> + inline cidrSchema

Schema updates (lib/schemas.ts):
  - sshKeySchema: removed public_key (derived server-side), added is_default: z.boolean()
  - gitRepoSchema: added auth_type, https_token, webhook_secret fields

Pattern applied:
  - useForm with zodResolver for validation
  - form.register() for text/number/select/checkbox inputs
  - valueAsNumber: true for number fields (priority, ssh_port)
  - form.handleSubmit wrapping async submit handlers
  - Inline validation errors via form.formState.errors.{field}.message
  - noValidate on <form> to disable browser validation
  - form.reset() on dialog close and success
  - Existing apiFetch + try/catch mutation pattern preserved (no useMutation yet)

Manual useState variables removed per file:
  - groups/new: name, description, priority
  - ssh-keys: keyName, privateKey, isDefault
  - git-repos: name, url, branch, authType, sshKeyId, httpsToken, webhookSecret
  - hosts/new: hostname, ipAddress, sshPort, sshKeyId, selectedGroups
  - hosts/discover: cidr (+ removed CIDR_REGEX constant, now in Zod schema)

Kept as useState (non-form state):
  - error/loading (API call state, to be replaced by useMutation in T20)
  - dialogOpen, editingRepo, searchQuery, phase, etc. (UI state)
